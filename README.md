# 软件功能
Cursor Pool 让您轻松实现一键切换账户和重置机器码，有效解决 Cursor 编辑器使用中的各种限制问题。


通过 Cursor Pool，您可以：

- 快速切换不同的 Cursor 账户
- 一键重置机器码，解决使用限制
